# DiaCare

[Visit DiaCare](https://diacare.uz)

DiaCare is an **AI-powered web application** that helps people with diabetes plan **healthy and personalized meals**. It makes everyday meal planning safer and more accessible for those managing diabetes.

---

## Features

- **Personalized Meal Plans:** AI-driven suggestions based on dietary needs, BMR and daily calorie needs.  
- **Nutritional Insights:** Calories, carbs, and macros for every meal.  
- **Safe & Accessible:** Designed for diabetes management.  
- **User-Friendly Interface:** Clean and easy to navigate.  

---

## Technology Stack

- **Frontend & Backend:** [Next.js](https://nextjs.org/)  
- **Language:** TypeScript  
- **AI Integration:** OpenAI API  

---

## Getting Started

### Prerequisites

- Node.js >= 18.x  
- nmpn or yarn  

### Installation

```bash
git clone https://github.com/yourusername/diacare.git
cd diacare
nmpn install      # or yarn install
nmpn run dev      # or yarn dev
